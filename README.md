## 👨‍💻 Sobre mim

Sou estudante de programação e atualmente estou aprendendo desenvolvimento backend com **Node.js** através da plataforma **DIO (Digital Innovation One)**.

Este projeto faz parte da minha evolução prática, aplicando conceitos de lógica, modularização e organização de código.

---

## 💻 Descrição do Projeto

Neste projeto, recriei a lógica básica de um carrinho de compras inspirado na Shopee.

O sistema permite:

- Adicionar itens ao carrinho
- Remover itens
- Deletar produtos
- Listar itens do carrinho
- Calcular o valor total automaticamente

---

## 🧠 Conceitos aplicados

- Separação de responsabilidades (services)

- Funções assíncronas (async/await)
